# Flexbox : propriétés des éléments flexibles

## Objectifs du cours

À la fin de ce cours, vous serez capables de :

- distinguer les propriétés appliquées au conteneur flexible et celles appliquées aux éléments flexibles ;
- modifier l’ordre d’affichage d’un élément avec `order` ;
- permettre à un élément de s’agrandir avec `flex-grow` ;
- permettre à un élément de rétrécir avec `flex-shrink` ;
- définir une taille de départ avec `flex-basis` ;
- utiliser le raccourci `flex` ;
- personnaliser l’alignement d’un seul élément avec `align-self`.

---

# 1. Mise en contexte

Jusqu’ici, nous avons surtout utilisé des propriétés appliquées au **conteneur flexible**.

Par exemple :

```css
.container {
  display: flex;
  justify-content: center;
  align-items: center;
  flex-wrap: wrap;
}
```

Ces propriétés contrôlent le comportement général des éléments à l’intérieur du conteneur.

Mais Flexbox permet aussi d’appliquer certaines propriétés directement aux **éléments flexibles**, c’est-à-dire aux enfants directs du conteneur.

---

# 2. Rappel : qu’est-ce qu’un élément flexible ?

Un élément flexible est un enfant direct d’un conteneur flexible.

Exemple :

```html
<div class="container">
  <div class="item">Élément 1</div>
  <div class="item">Élément 2</div>
  <div class="item">Élément 3</div>
</div>
```

```css
.container {
  display: flex;
}
```

Ici :

- `.container` est le conteneur flexible ;
- les trois `.item` sont les éléments flexibles.

Les propriétés que nous allons voir dans ce cours s’appliquent donc aux éléments `.item`.

---

# 3. Propriétés du conteneur vs propriétés des éléments

Il est important de distinguer deux catégories de propriétés Flexbox.

## 3.1 Propriétés appliquées au conteneur

Ces propriétés contrôlent l’ensemble de la disposition.

Exemples :

```css
.container {
  display: flex;
  flex-direction: row;
  justify-content: center;
  align-items: center;
  flex-wrap: wrap;
}
```

Elles répondent à des questions comme :

- Dans quelle direction les éléments sont-ils placés ?
- Les éléments peuvent-ils passer à la ligne ?
- Comment les éléments sont-ils alignés ?
- Comment l’espace est-il réparti ?

---

## 3.2 Propriétés appliquées aux éléments flexibles

Ces propriétés contrôlent un élément en particulier.

Exemples :

```css
.item {
  order: 1;
  flex-grow: 1;
  flex-shrink: 1;
  flex-basis: 200px;
  align-self: center;
}
```

Elles répondent à des questions comme :

- Cet élément doit-il s’afficher avant ou après les autres ?
- Cet élément peut-il s’agrandir ?
- Cet élément peut-il rétrécir ?
- Quelle est sa taille de départ ?
- Cet élément doit-il avoir un alignement différent des autres ?

---

# 4. Modifier l’ordre d’affichage avec `order`

Par défaut, les éléments HTML s’affichent dans l’ordre dans lequel ils sont écrits dans le HTML.

Exemple :

```html
<div class="container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
</div>
```

Par défaut, l’affichage sera :

```text
1  2  3
```

La propriété `order` permet de modifier l’ordre visuel des éléments flexibles.

---

## 4.1 Valeur par défaut de `order`

La valeur par défaut de `order` est :

```css
order: 0;
```

Cela signifie que tous les éléments ont le même ordre au départ.

Exemple :

```css
.item {
  order: 0;
}
```

Si tous les éléments ont le même `order`, ils restent dans leur ordre HTML normal.

---

## 4.2 Utiliser une valeur négative

Un élément avec une valeur plus petite sera affiché avant les autres.

Exemple :

```css
.item-important {
  order: -1;
}
```

HTML :

```html
<div class="container">
  <div class="item">1</div>
  <div class="item item-important">2</div>
  <div class="item">3</div>
</div>
```

Même si l’élément `2` est écrit en deuxième dans le HTML, il sera affiché avant les autres.

Résultat visuel :

```text
2  1  3
```

---

## 4.3 Utiliser une valeur positive

Un élément avec une valeur plus grande sera affiché après les autres.

Exemple :

```css
.item-last {
  order: 1;
}
```

HTML :

```html
<div class="container">
  <div class="item item-last">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
</div>
```

Résultat visuel :

```text
2  3  1
```

L’élément `1` est affiché à la fin, car son `order` est plus élevé que celui des autres éléments.

---

## 4.4 Point d’attention avec `order`

La propriété `order` change seulement l’ordre visuel.

Elle ne change pas l’ordre réel dans le HTML.

Il faut donc l’utiliser avec prudence, surtout pour l’accessibilité et la navigation au clavier.

En général, il est préférable que l’ordre du HTML reste logique.

---

# 5. Permettre à un élément de s’agrandir avec `flex-grow`

La propriété `flex-grow` indique si un élément flexible peut s’agrandir pour utiliser l’espace disponible sur l’axe principal.

Elle s’applique à un élément flexible.

```css
.item {
  flex-grow: 1;
}
```

---

## 5.1 Valeur par défaut de `flex-grow`

La valeur par défaut est :

```css
flex-grow: 0;
```

Cela signifie que, par défaut, un élément ne s’agrandit pas automatiquement pour occuper l’espace disponible.

Exemple :

```css
.container {
  display: flex;
}

.item {
  width: 100px;
  flex-grow: 0;
}
```

Même s’il reste de l’espace libre dans le conteneur, les éléments gardent leur taille.

---

## 5.2 Tous les éléments avec `flex-grow: 1`

Si tous les éléments ont :

```css
flex-grow: 1;
```

alors l’espace disponible est partagé équitablement entre eux.

Exemple :

```css
.container {
  display: flex;
}

.item {
  flex-grow: 1;
}
```

Si le conteneur est large, les éléments vont s’étirer pour remplir l’espace disponible.

Résultat logique :

```text
[ Élément 1 ][ Élément 2 ][ Élément 3 ]
```

Chaque élément reçoit une part équivalente de l’espace.

---

## 5.3 Un élément avec `flex-grow: 2`

Si un élément a une valeur plus grande, il reçoit une plus grande part de l’espace disponible.

Exemple :

```css
.item {
  flex-grow: 1;
}

.item-large {
  flex-grow: 2;
}
```

HTML :

```html
<div class="container">
  <div class="item">1</div>
  <div class="item item-large">2</div>
  <div class="item">3</div>
</div>
```

Ici, l’élément `2` prendra deux fois plus d’espace disponible que les éléments qui ont `flex-grow: 1`.

Résultat logique :

```text
[ 1 ][     2     ][ 3 ]
```

---

# 6. Permettre à un élément de rétrécir avec `flex-shrink`

La propriété `flex-shrink` indique si un élément flexible peut rétrécir lorsqu’il n’y a pas assez d’espace dans le conteneur.

Elle s’applique à un élément flexible.

```css
.item {
  flex-shrink: 1;
}
```

---

## 6.1 Valeur par défaut de `flex-shrink`

La valeur par défaut est :

```css
flex-shrink: 1;
```

Cela signifie que, par défaut, les éléments flexibles peuvent rétrécir si l’espace est insuffisant.

Exemple :

```css
.container {
  display: flex;
  width: 400px;
}

.item {
  width: 200px;
  flex-shrink: 1;
}
```

Si on a trois éléments de `200px`, il faudrait normalement `600px`.

Mais le conteneur fait seulement `400px`.

Flexbox va donc essayer de réduire les éléments pour les faire entrer dans le conteneur.

---

## 6.2 Empêcher un élément de rétrécir

Pour empêcher un élément de rétrécir, on peut utiliser :

```css
.item {
  flex-shrink: 0;
}
```

Cela signifie que l’élément conserve sa taille autant que possible.

Exemple :

```css
.sidebar {
  width: 250px;
  flex-shrink: 0;
}
```

Cette technique est utile pour une barre latérale qui doit garder une largeur fixe.

---

## 6.3 Un élément avec `flex-shrink: 2`

Si un élément a une valeur plus grande, il rétrécira plus rapidement que les autres.

Exemple :

```css
.item {
  flex-shrink: 1;
}

.item-small {
  flex-shrink: 2;
}
```

Dans cet exemple, `.item-small` rétrécira plus que les éléments qui ont `flex-shrink: 1`.

---

# 7. Définir la taille initiale avec `flex-basis`

La propriété `flex-basis` définit la taille initiale d’un élément flexible avant que l’espace disponible soit réparti.

Elle s’applique à un élément flexible.

```css
.item {
  flex-basis: 100px;
}
```

On peut la voir comme une taille de départ.

---

## 7.1 Valeur par défaut de `flex-basis`

La valeur par défaut est :

```css
flex-basis: auto;
```

Avec `auto`, le navigateur utilise :

- la taille définie avec `width` ou `height`, si elle existe ;
- sinon, la taille du contenu.

Exemple :

```css
.item {
  width: 200px;
  flex-basis: auto;
}
```

Ici, la base de l’élément sera `200px`.

---

## 7.2 Utiliser une valeur précise

On peut définir une taille initiale précise.

```css
.item {
  flex-basis: 150px;
}
```

Cela signifie que l’élément commence avec une base de `150px`.

Ensuite, selon `flex-grow` et `flex-shrink`, il pourra s’agrandir ou rétrécir.

---

## 7.3 Différence entre `width` et `flex-basis`

Dans un contexte Flexbox, `flex-basis` sert de taille de référence sur l’axe principal.

Si l’axe principal est horizontal, `flex-basis` ressemble à une largeur de départ.

Si l’axe principal est vertical, `flex-basis` ressemble à une hauteur de départ.

Exemple :

```css
.container {
  display: flex;
  flex-direction: row;
}

.item {
  flex-basis: 200px;
}
```

Ici, `flex-basis` agit comme une largeur de départ.

Avec :

```css
.container {
  display: flex;
  flex-direction: column;
}

.item {
  flex-basis: 200px;
}
```

Ici, `flex-basis` agit plutôt comme une hauteur de départ.

---

# 8. Le raccourci `flex`

La propriété `flex` permet de définir trois propriétés en une seule ligne :

```css
flex-grow
flex-shrink
flex-basis
```

L’ordre est le suivant :

```css
flex: flex-grow flex-shrink flex-basis;
```

Exemple :

```css
.item {
  flex: 1 1 200px;
}
```

Ce code signifie :

```css
.item {
  flex-grow: 1;
  flex-shrink: 1;
  flex-basis: 200px;
}
```

---

# 9. Valeur par défaut de `flex`

Par défaut, un élément flexible se comporte comme ceci :

```css
.item {
  flex: 0 1 auto;
}
```

Cela signifie :

```css
.item {
  flex-grow: 0;
  flex-shrink: 1;
  flex-basis: auto;
}
```

Donc, par défaut :

- l’élément ne s’agrandit pas automatiquement ;
- l’élément peut rétrécir ;
- sa taille de départ est automatique.

---

# 10. Le mot-clé `flex: auto`

Le mot-clé :

```css
flex: auto;
```

est équivalent à :

```css
.item {
  flex: 1 1 auto;
}
```

Cela signifie :

- l’élément peut s’agrandir ;
- l’élément peut rétrécir ;
- sa taille de départ est automatique.

Cette valeur permet à l’élément de s’adapter à l’espace disponible.

---

# 11. Le mot-clé `flex: none`

Le mot-clé :

```css
flex: none;
```

est équivalent à :

```css
.item {
  flex: 0 0 auto;
}
```

Cela signifie :

- l’élément ne s’agrandit pas ;
- l’élément ne rétrécit pas ;
- sa taille de départ est automatique.

Cette valeur est utile lorsqu’on veut qu’un élément garde sa taille naturelle.

---

# 12. Exemple pratique : mise en page avec une barre latérale

HTML :

```html
<main class="layout">
  <aside class="sidebar">Menu</aside>
  <section class="content">Contenu principal</section>
</main>
```

CSS :

```css
.layout {
  display: flex;
  min-height: 100vh;
}

.sidebar {
  flex: 0 0 250px;
  border-right: 1px solid black;
  padding: 1rem;
}

.content {
  flex: 1 1 auto;
  padding: 1rem;
}
```

Explication :

```css
.sidebar {
  flex: 0 0 250px;
}
```

La barre latérale :

- ne s’agrandit pas ;
- ne rétrécit pas ;
- commence avec une largeur de `250px`.

```css
.content {
  flex: 1 1 auto;
}
```

Le contenu principal :

- peut s’agrandir ;
- peut rétrécir ;
- utilise l’espace restant.

---

# 13. Exemple pratique : cartes qui partagent l’espace

HTML :

```html
<section class="cards">
  <article class="card">Carte 1</article>
  <article class="card">Carte 2</article>
  <article class="card">Carte 3</article>
</section>
```

CSS :

```css
.cards {
  display: flex;
  gap: 1rem;
}

.card {
  flex: 1;
  padding: 1rem;
  border: 1px solid black;
}
```

Ici :

```css
.card {
  flex: 1;
}
```

permet aux cartes de partager l’espace disponible.

Dans ce cas, les cartes auront une largeur équivalente.

---

# 14. Comprendre `flex: 1`

On voit très souvent :

```css
.item {
  flex: 1;
}
```

Cette écriture signifie généralement que l’élément peut prendre une part de l’espace disponible.

Dans la pratique, cela permet souvent à plusieurs éléments de se partager l’espace.

Exemple :

```css
.card {
  flex: 1;
}
```

Si trois cartes ont `flex: 1`, elles occupent chacune une part équivalente de l’espace disponible.

---

# 15. Définir l’alignement particulier d’un élément avec `align-self`

La propriété `align-self` permet de modifier l’alignement d’un seul élément flexible sur l’axe secondaire.

Elle permet à un élément de passer par-dessus la valeur définie par `align-items` sur le conteneur.

---

## 15.1 Exemple avec `align-items`

```css
.container {
  display: flex;
  align-items: flex-start;
}
```

Dans cet exemple, tous les éléments sont alignés au début de l’axe secondaire.

Mais on peut décider qu’un élément précis sera aligné différemment.

---

## 15.2 Exemple avec `align-self`

HTML :

```html
<div class="container">
  <div class="item">1</div>
  <div class="item special">2</div>
  <div class="item">3</div>
</div>
```

CSS :

```css
.container {
  display: flex;
  align-items: flex-start;
  height: 200px;
}

.special {
  align-self: flex-end;
}
```

Dans cet exemple :

- les éléments `1` et `3` sont alignés au début de l’axe secondaire ;
- l’élément `2` est aligné à la fin de l’axe secondaire.

---

## 15.3 Valeurs possibles de `align-self`

Les valeurs de `align-self` sont les mêmes que celles de `align-items`.

| Valeur       | Effet                                          |
| ------------ | ---------------------------------------------- |
| `auto`       | utilise la valeur de `align-items`             |
| `stretch`    | étire l’élément                                |
| `flex-start` | place l’élément au début de l’axe secondaire   |
| `flex-end`   | place l’élément à la fin de l’axe secondaire   |
| `center`     | centre l’élément sur l’axe secondaire          |
| `baseline`   | aligne l’élément sur la ligne de base du texte |

La valeur par défaut est :

```css
align-self: auto;
```

Cela signifie que l’élément suit normalement la règle `align-items` du conteneur.

---

# 16. Exemple complet

HTML :

```html
<section class="container">
  <article class="card first">Carte 1</article>
  <article class="card second">Carte 2</article>
  <article class="card third">Carte 3</article>
</section>
```

CSS :

```css
.container {
  display: flex;
  align-items: flex-start;
  gap: 1rem;
  height: 250px;
  border: 1px solid black;
  padding: 1rem;
}

.card {
  flex: 1 1 150px;
  padding: 1rem;
  border: 1px solid black;
}

.first {
  order: 2;
}

.second {
  order: -1;
  flex-grow: 2;
}

.third {
  align-self: flex-end;
}
```

Explication :

- `.container` est le conteneur flexible ;
- `.card` représente les éléments flexibles ;
- `flex: 1 1 150px` donne une base de `150px`, tout en permettant aux cartes de grandir et de rétrécir ;
- `.first` a `order: 2`, donc elle sera affichée plus tard ;
- `.second` a `order: -1`, donc elle sera affichée avant les autres ;
- `.second` a aussi `flex-grow: 2`, donc elle prendra plus d’espace disponible ;
- `.third` utilise `align-self: flex-end`, donc elle est alignée différemment des autres sur l’axe secondaire.

---

# 17. Points d’attention

## 17.1 Les propriétés des éléments ne remplacent pas celles du conteneur

Pour utiliser `order`, `flex-grow`, `flex-shrink`, `flex-basis`, `flex` ou `align-self`, il faut que l’élément soit dans un conteneur flexible.

Autrement dit, son parent doit avoir :

```css
display: flex;
```

---

## 17.2 `order` modifie l’affichage, pas le HTML

La propriété `order` ne déplace pas réellement l’élément dans le code HTML.

Elle modifie seulement son ordre visuel.

Il faut donc éviter de l’utiliser pour corriger une mauvaise structure HTML.

---

## 17.3 `flex-grow` partage l’espace disponible

`flex-grow` ne définit pas directement une largeur fixe.

Il indique comment l’espace libre doit être partagé.

Deux éléments avec `flex-grow: 1` se partagent l’espace.

Un élément avec `flex-grow: 2` reçoit une part plus grande.

---

## 17.4 `flex-shrink` agit quand l’espace manque

`flex-shrink` devient important lorsque les éléments n’ont pas assez de place.

Par défaut, les éléments peuvent rétrécir, car la valeur est :

```css
flex-shrink: 1;
```

Pour empêcher un élément de rétrécir :

```css
flex-shrink: 0;
```

---

## 17.5 `flex-basis` dépend de l’axe principal

`flex-basis` représente une taille de départ sur l’axe principal.

Avec `flex-direction: row`, elle correspond à une largeur de départ.

Avec `flex-direction: column`, elle correspond à une hauteur de départ.

---

## 17.6 `align-self` s’applique à un seul élément

`align-items` s’applique au conteneur et influence tous les éléments.

`align-self` s’applique à un élément précis et permet de lui donner un alignement particulier.

---

# 18. Mini-bilan

Flexbox permet de contrôler la disposition globale avec des propriétés appliquées au conteneur.

Mais il permet aussi de contrôler certains éléments individuellement.

La propriété `order` permet de modifier l’ordre visuel d’un élément.

```css
.item {
  order: -1;
}
```

La propriété `flex-grow` permet à un élément de s’agrandir.

```css
.item {
  flex-grow: 1;
}
```

La propriété `flex-shrink` permet à un élément de rétrécir.

```css
.item {
  flex-shrink: 1;
}
```

La propriété `flex-basis` définit une taille de départ.

```css
.item {
  flex-basis: 200px;
}
```

La propriété `flex` permet de combiner les trois.

```css
.item {
  flex: 1 1 200px;
}
```

La propriété `align-self` permet d’aligner un élément différemment des autres.

```css
.item {
  align-self: center;
}
```

# Cheatsheet — Flexbox : propriétés des éléments flexibles

## Propriétés principales

| Propriété     | Où l’utiliser ?         | Rôle                                     |
| ------------- | ----------------------- | ---------------------------------------- |
| `order`       | sur un élément flexible | modifie l’ordre visuel                   |
| `flex-grow`   | sur un élément flexible | permet à l’élément de s’agrandir         |
| `flex-shrink` | sur un élément flexible | permet à l’élément de rétrécir           |
| `flex-basis`  | sur un élément flexible | définit la taille initiale               |
| `flex`        | sur un élément flexible | raccourci pour `grow`, `shrink`, `basis` |
| `align-self`  | sur un élément flexible | modifie l’alignement d’un seul élément   |

---

## Valeurs importantes

```css
.item {
  order: 0;
}
```

Ordre par défaut.

```css
.item {
  flex-grow: 0;
}
```

L’élément ne s’agrandit pas automatiquement.

```css
.item {
  flex-shrink: 1;
}
```

L’élément peut rétrécir si l’espace manque.

```css
.item {
  flex-basis: auto;
}
```

La taille de départ est automatique.

```css
.item {
  flex: 0 1 auto;
}
```

Comportement par défaut complet.

---

## Raccourcis utiles

```css
.item {
  flex: auto;
}
```

Équivaut à :

```css
.item {
  flex: 1 1 auto;
}
```

L’élément peut grandir, peut rétrécir et utilise une base automatique.

```css
.item {
  flex: none;
}
```

Équivaut à :

```css
.item {
  flex: 0 0 auto;
}
```

L’élément ne grandit pas, ne rétrécit pas et garde sa taille automatique.

```css
.item {
  flex: 1;
}
```

Très utilisé pour permettre à plusieurs éléments de partager l’espace disponible.

---

## À retenir

Les propriétés comme `justify-content`, `align-items` et `flex-wrap` se mettent sur le conteneur.

Les propriétés comme `order`, `flex-grow`, `flex-shrink`, `flex-basis`, `flex` et `align-self` se mettent sur les éléments flexibles.

Exemple général :

```css
.container {
  display: flex;
}

.item {
  flex: 1 1 200px;
}

.item-special {
  order: -1;
  align-self: center;
}
```

Pour bien utiliser Flexbox, il faut toujours identifier :

```text
1. Quel est le conteneur flexible ?
2. Quels sont les enfants directs ?
3. Est-ce que je veux contrôler le groupe ou un élément précis ?
```
