# Flexbox : alignement et espacement des éléments

## Objectifs du cours

À la fin de ce cours, vous serez capables de :

- comprendre le rôle de l’axe principal et de l’axe secondaire dans Flexbox ;
- utiliser `justify-content` pour aligner les éléments sur l’axe principal ;
- utiliser `align-items` pour aligner les éléments sur l’axe secondaire ;
- distinguer `align-items` et `align-content` ;
- utiliser `align-content` lorsque le conteneur contient plusieurs lignes ;
- choisir la bonne propriété selon le type d’alignement souhaité.

---

# 1. Mise en contexte

Dans le cours précédent, nous avons vu que Flexbox permet de transformer un conteneur en conteneur flexible avec :

```css
.container {
  display: flex;
}
```

Nous avons aussi vu que les éléments flexibles sont organisés selon deux axes :

- l’axe principal ;
- l’axe secondaire.

Ces deux axes sont essentiels pour comprendre l’alignement avec Flexbox.

---

# 2. Rappel : axe principal et axe secondaire

L’axe principal est défini par la propriété `flex-direction`.

Par défaut, Flexbox utilise :

```css
flex-direction: row;
```

Cela signifie que les éléments sont placés horizontalement, de gauche à droite.

Dans ce cas :

```text
Axe principal : horizontal
Axe secondaire : vertical
```

Si on utilise plutôt :

```css
flex-direction: column;
```

Alors les éléments sont placés verticalement, de haut en bas.

Dans ce cas :

```text
Axe principal : vertical
Axe secondaire : horizontal
```

Il faut donc toujours se demander :

```text
Dans quelle direction mes éléments sont-ils placés ?
```

Cette question permet de savoir quelle propriété utiliser.

---

# 3. Aligner les éléments sur l’axe principal avec `justify-content`

La propriété `justify-content` permet de gérer l’alignement et l’espacement des éléments sur l’axe principal.

Elle s’utilise sur le conteneur flexible.

```css
.container {
  display: flex;
  justify-content: flex-start;
}
```

La valeur par défaut est :

```css
justify-content: flex-start;
```

Cela signifie que les éléments sont placés au début de l’axe principal.

---

# 4. Les valeurs de `justify-content`

La propriété `justify-content` peut prendre plusieurs valeurs.

| Valeur          | Effet                                          |
| --------------- | ---------------------------------------------- |
| `flex-start`    | place les éléments au début de l’axe principal |
| `flex-end`      | place les éléments à la fin de l’axe principal |
| `center`        | centre les éléments sur l’axe principal        |
| `space-between` | répartit l’espace entre les éléments           |
| `space-around`  | ajoute de l’espace autour de chaque élément    |
| `space-evenly`  | répartit l’espace de manière égale partout     |

---

# 5. `justify-content: flex-start`

La valeur `flex-start` place les éléments au début de l’axe principal.

```css
.container {
  display: flex;
  justify-content: flex-start;
}
```

Si l’axe principal est horizontal, les éléments seront placés à gauche.

```text
[1][2][3]----------------
```

C’est la valeur par défaut.

---

# 6. `justify-content: flex-end`

La valeur `flex-end` place les éléments à la fin de l’axe principal.

```css
.container {
  display: flex;
  justify-content: flex-end;
}
```

Si l’axe principal est horizontal, les éléments seront placés à droite.

```text
----------------[1][2][3]
```

Cette valeur est utile lorsqu’on veut pousser des éléments vers la fin d’une ligne.

---

# 7. `justify-content: center`

La valeur `center` centre les éléments sur l’axe principal.

```css
.container {
  display: flex;
  justify-content: center;
}
```

Résultat logique :

```text
--------[1][2][3]--------
```

Cette valeur est souvent utilisée pour centrer une navigation, des boutons ou des cartes.

---

# 8. `justify-content: space-between`

La valeur `space-between` répartit l’espace libre entre les éléments.

Le premier élément est collé au début de l’axe principal.

Le dernier élément est collé à la fin de l’axe principal.

L’espace restant est réparti entre les éléments.

```css
.container {
  display: flex;
  justify-content: space-between;
}
```

Résultat logique :

```text
[1]--------[2]--------[3]
```

Cette valeur est très utile pour créer une barre de navigation avec un élément à gauche et un élément à droite.

Exemple :

```html
<nav class="navbar">
  <div class="logo">Logo</div>
  <div class="links">Liens</div>
</nav>
```

```css
.navbar {
  display: flex;
  justify-content: space-between;
}
```

---

# 9. `justify-content: space-around`

La valeur `space-around` ajoute de l’espace autour de chaque élément.

Chaque élément reçoit une unité d’espace à gauche et une unité d’espace à droite.

Cela signifie que l’espace entre deux éléments est souvent plus grand que l’espace sur les bords.

```css
.container {
  display: flex;
  justify-content: space-around;
}
```

Résultat logique :

```text
----[1]--------[2]--------[3]----
```

Pourquoi l’espace entre les éléments est-il plus grand ?

Parce que chaque élément possède de l’espace de chaque côté.

Entre deux éléments, les espaces s’additionnent.

---

# 10. `justify-content: space-evenly`

La valeur `space-evenly` répartit l’espace de manière parfaitement égale.

L’espace avant le premier élément, entre les éléments et après le dernier élément est identique.

```css
.container {
  display: flex;
  justify-content: space-evenly;
}
```

Résultat logique :

```text
----[1]----[2]----[3]----
```

Cette valeur est pratique lorsqu’on veut une répartition très régulière.

---

# 11. Exemple complet avec `justify-content`

HTML :

```html
<section class="container">
  <div class="item">Un</div>
  <div class="item">Deux</div>
  <div class="item">Trois</div>
</section>
```

CSS :

```css
.container {
  display: flex;
  justify-content: center;
  border: 1px solid black;
  padding: 1rem;
}

.item {
  padding: 1rem;
  border: 1px solid black;
}
```

Dans cet exemple :

- `.container` est le conteneur flexible ;
- les trois `div` sont les éléments flexibles ;
- `justify-content: center` centre les éléments sur l’axe principal.

---

# 12. Aligner les éléments sur l’axe secondaire avec `align-items`

La propriété `align-items` permet d’aligner les éléments sur l’axe secondaire.

Elle s’utilise sur le conteneur flexible.

```css
.container {
  display: flex;
  align-items: flex-start;
}
```

Si les éléments sont placés horizontalement avec `flex-direction: row`, alors l’axe secondaire est vertical.

Dans ce cas, `align-items` contrôle l’alignement vertical des éléments.

---

# 13. Les valeurs de `align-items`

La propriété `align-items` peut prendre plusieurs valeurs.

| Valeur       | Effet                                               |
| ------------ | --------------------------------------------------- |
| `stretch`    | étire les éléments dans l’espace disponible         |
| `flex-start` | place les éléments au début de l’axe secondaire     |
| `flex-end`   | place les éléments à la fin de l’axe secondaire     |
| `center`     | centre les éléments sur l’axe secondaire            |
| `baseline`   | aligne les éléments selon la ligne de base du texte |

La valeur par défaut est :

```css
align-items: stretch;
```

---

# 14. `align-items: stretch`

La valeur `stretch` étire les éléments dans l’espace disponible sur l’axe secondaire.

```css
.container {
  display: flex;
  align-items: stretch;
}
```

C’est la valeur par défaut.

Si le conteneur a une hauteur définie, les éléments peuvent s’étirer pour occuper cette hauteur.

Exemple :

```css
.container {
  display: flex;
  height: 200px;
  align-items: stretch;
}
```

Les éléments seront étirés verticalement si l’axe principal est horizontal.

Attention : `stretch` respecte les contraintes comme :

```css
height
width
min-height
max-height
min-width
max-width
```

---

# 15. `align-items: flex-start`

La valeur `flex-start` place les éléments au début de l’axe secondaire.

```css
.container {
  display: flex;
  align-items: flex-start;
}
```

Si l’axe secondaire est vertical, les éléments seront placés en haut du conteneur.

```text
[1][2][3]
---------
---------
```

---

# 16. `align-items: flex-end`

La valeur `flex-end` place les éléments à la fin de l’axe secondaire.

```css
.container {
  display: flex;
  align-items: flex-end;
}
```

Si l’axe secondaire est vertical, les éléments seront placés en bas du conteneur.

```text
---------
---------
[1][2][3]
```

---

# 17. `align-items: center`

La valeur `center` centre les éléments sur l’axe secondaire.

```css
.container {
  display: flex;
  align-items: center;
}
```

Si l’axe secondaire est vertical, les éléments seront centrés verticalement.

```text
---------
[1][2][3]
---------
```

Cette valeur est très souvent utilisée avec `justify-content: center` pour centrer un élément au milieu d’un conteneur.

Exemple :

```css
.container {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 100vh;
}
```

Dans cet exemple, l’élément est centré horizontalement et verticalement.

---

# 18. `align-items: baseline`

La valeur `baseline` aligne les éléments selon la ligne de base du texte.

```css
.container {
  display: flex;
  align-items: baseline;
}
```

Cette valeur est surtout utile lorsque les éléments contiennent du texte de tailles différentes.

Exemple :

```html
<div class="container">
  <p class="small">Petit texte</p>
  <p class="large">Grand texte</p>
</div>
```

```css
.container {
  display: flex;
  align-items: baseline;
}

.small {
  font-size: 1rem;
}

.large {
  font-size: 2rem;
}
```

Les textes seront alignés selon leur ligne de base, plutôt que simplement en haut, en bas ou au centre.

---

# 19. Relation entre `justify-content` et `align-items`

Les deux propriétés servent à aligner les éléments, mais elles ne travaillent pas sur le même axe.

| Propriété         | Axe concerné   |
| ----------------- | -------------- |
| `justify-content` | axe principal  |
| `align-items`     | axe secondaire |

Avec :

```css
flex-direction: row;
```

on obtient :

```text
justify-content → horizontal
align-items     → vertical
```

Avec :

```css
flex-direction: column;
```

on obtient :

```text
justify-content → vertical
align-items     → horizontal
```

C’est une des notions les plus importantes à retenir.

---

# 20. Centrer un élément avec Flexbox

Un cas très courant consiste à centrer un élément horizontalement et verticalement.

HTML :

```html
<section class="page">
  <div class="card">Carte centrée</div>
</section>
```

CSS :

```css
.page {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
}

.card {
  padding: 2rem;
  border: 1px solid black;
}
```

Explication :

- `display: flex` active Flexbox ;
- `justify-content: center` centre sur l’axe principal ;
- `align-items: center` centre sur l’axe secondaire ;
- `min-height: 100vh` permet à la section d’occuper au moins toute la hauteur visible de l’écran.

---

# 21. Aligner plusieurs lignes avec `align-content`

La propriété `align-content` sert à gérer l’espacement entre plusieurs lignes d’éléments flexibles.

Elle s’utilise seulement si le conteneur contient plusieurs lignes.

Pour avoir plusieurs lignes, il faut généralement utiliser :

```css
flex-wrap: wrap;
```

Exemple :

```css
.container {
  display: flex;
  flex-wrap: wrap;
  align-content: center;
}
```

Si les éléments restent tous sur une seule ligne, `align-content` n’aura généralement pas d’effet visible.

---

# 22. Différence entre `align-items` et `align-content`

Il est important de ne pas confondre `align-items` et `align-content`.

| Propriété       | Rôle                                          |
| --------------- | --------------------------------------------- |
| `align-items`   | aligne les éléments à l’intérieur d’une ligne |
| `align-content` | aligne ou espace les lignes entre elles       |

Donc :

- `align-items` agit sur les éléments ;
- `align-content` agit sur les lignes.

---

# 23. Les valeurs de `align-content`

La propriété `align-content` peut prendre plusieurs valeurs.

| Valeur          | Effet                                         |
| --------------- | --------------------------------------------- |
| `stretch`       | étire les lignes dans l’espace disponible     |
| `flex-start`    | place les lignes au début de l’axe secondaire |
| `flex-end`      | place les lignes à la fin de l’axe secondaire |
| `center`        | centre les lignes                             |
| `space-between` | répartit l’espace entre les lignes            |
| `space-around`  | ajoute de l’espace autour des lignes          |

La valeur par défaut est :

```css
align-content: stretch;
```

---

# 24. `align-content: flex-start`

La valeur `flex-start` place les lignes au début de l’axe secondaire.

```css
.container {
  display: flex;
  flex-wrap: wrap;
  align-content: flex-start;
}
```

Les lignes seront regroupées au début du conteneur.

---

# 25. `align-content: flex-end`

La valeur `flex-end` place les lignes à la fin de l’axe secondaire.

```css
.container {
  display: flex;
  flex-wrap: wrap;
  align-content: flex-end;
}
```

Les lignes seront regroupées vers la fin du conteneur.

---

# 26. `align-content: center`

La valeur `center` centre les lignes sur l’axe secondaire.

```css
.container {
  display: flex;
  flex-wrap: wrap;
  align-content: center;
}
```

Les lignes seront regroupées au centre du conteneur.

---

# 27. `align-content: space-between`

La valeur `space-between` répartit l’espace libre entre les lignes.

```css
.container {
  display: flex;
  flex-wrap: wrap;
  align-content: space-between;
}
```

La première ligne est placée au début.

La dernière ligne est placée à la fin.

L’espace restant est réparti entre les lignes.

---

# 28. `align-content: space-around`

La valeur `space-around` ajoute de l’espace autour de chaque ligne.

```css
.container {
  display: flex;
  flex-wrap: wrap;
  align-content: space-around;
}
```

Chaque ligne reçoit de l’espace avant et après elle.

---

# 29. `align-content: stretch`

La valeur `stretch` étire les lignes dans l’espace disponible.

```css
.container {
  display: flex;
  flex-wrap: wrap;
  align-content: stretch;
}
```

C’est la valeur par défaut.

- `align-items: center` aligne verticalement les éléments ;
- `.menu` devient aussi un conteneur flexible pour aligner les liens.

# 34. Points d’attention

## 34.1 `justify-content` dépend de l’axe principal

Il ne faut pas penser que `justify-content` veut toujours dire horizontal.

Cela dépend de `flex-direction`.

Avec :

```css
flex-direction: row;
```

`justify-content` agit horizontalement.

Avec :

```css
flex-direction: column;
```

`justify-content` agit verticalement.

---

## 34.2 `align-items` dépend de l’axe secondaire

Il ne faut pas penser que `align-items` veut toujours dire vertical.

Cela dépend aussi de `flex-direction`.

Avec :

```css
flex-direction: row;
```

`align-items` agit verticalement.

Avec :

```css
flex-direction: column;
```

`align-items` agit horizontalement.

---

## 34.3 `align-content` nécessite plusieurs lignes

La propriété `align-content` n’est utile que lorsqu’il y a plusieurs lignes d’éléments.

Il faut donc généralement avoir :

```css
flex-wrap: wrap;
```

et assez d’éléments pour créer plusieurs lignes.

---

## 34.4 La hauteur du conteneur influence l’effet visible

Pour voir clairement l’effet de `align-items` ou `align-content`, le conteneur doit souvent avoir une hauteur définie.

Exemple :

```css
.container {
  height: 300px;
}
```

Sans hauteur définie, il peut être difficile de voir l’effet de certaines propriétés.

---

## 34.5 `gap` est souvent plus simple que les marges

Pour créer de l’espace entre les éléments flexibles, on peut utiliser `gap`.

Exemple :

```css
.container {
  display: flex;
  gap: 1rem;
}
```

Cela évite souvent d’ajouter des marges manuellement sur chaque élément.

---

# 35. Mini-bilan

Flexbox permet non seulement de placer les éléments, mais aussi de gérer leur alignement et leur espacement.

La propriété `justify-content` agit sur l’axe principal.

```css
justify-content: center;
```

La propriété `align-items` agit sur l’axe secondaire.

```css
align-items: center;
```

La propriété `align-content` agit sur l’espacement entre plusieurs lignes.

```css
align-content: space-between;
```

Pour bien utiliser ces propriétés, il faut toujours commencer par identifier :

```text
1. Quelle est ma direction ?
2. Quel est mon axe principal ?
3. Quel est mon axe secondaire ?
4. Est-ce que j’aligne des éléments ou des lignes ?
```

## À retenir

```css
.container {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

Ce code est souvent utilisé pour centrer un élément horizontalement et verticalement.

Mais il faut retenir que :

```text
justify-content suit l’axe principal.
align-items suit l’axe secondaire.
```

L’axe principal dépend de :

```css

```
