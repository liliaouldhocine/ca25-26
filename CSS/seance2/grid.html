Guide Complet du CSS Grid

Apprenez progressivement à utiliser CSS Grid à travers des exemples interactifs

1. Qu'est-ce que CSS Grid ?

CSS Grid est un système de mise en page bidimensionnel conçu pour le web. Il vous permet de diviser une page en régions principales ou de définir des relations en termes de taille, position et couche entre éléments HTML.

.conteneur {
  display: grid;
}

Cette simple déclaration transforme un élément en conteneur grid et ses enfants directs en éléments grid.

2. Conteneur Grid et Éléments Grid

Le système Grid se compose de deux types d'éléments : le conteneur grid (parent) et les éléments grid (enfants).

3. Définir les colonnes et lignes (grid-template-columns/rows)

Les propriétés grid-template-columns et grid-template-rows définissent les colonnes et lignes du grid.

1
2
3
4
5
6
.conteneur {
  display: grid;
  grid-template-columns: none;
  grid-template-rows: none;
}

4. Espacement entre les éléments (gap)

La propriété gap définit l'espacement entre les lignes et les colonnes.

1
2
3
4
5
6
.conteneur {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}

5. Placement des éléments (grid-column et grid-row)

Les propriétés grid-column et grid-row permettent de positionner les éléments sur des lignes spécifiques.

1
2
3
4
5
6
.item1 {
  grid-column: auto;
  grid-row: auto;
}

6. Zones de grid (grid-template-areas)

La propriété grid-template-areas permet de nommer des zones dans le grid et d'y placer les éléments.

Header
Sidebar
Content
Footer
.conteneur {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-areas: none;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.content { grid-area: content; }
.footer { grid-area: footer; }

7. Alignement dans le grid (align-items, justify-items)

Les propriétés d'alignement contrôlent le positionnement des éléments à l'intérieur de leur cellule grid.

1
2
3
4
5
6
.conteneur {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  align-items: stretch;
  justify-items: stretch;
}

8. Exemple pratique : Mise en page complète

Voici un exemple pratique d'une mise en page complète utilisant CSS Grid :

Header
Sidebar
Content
Footer
.conteneur {
  display: grid;
  grid-template-columns: 1fr 3fr;
  grid-template-rows: 80px 1fr 80px;
  grid-template-areas:
    'header header'
    'sidebar content'
    'footer footer';
  gap: 15px;
  height: 400px;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.content { grid-area: content; }
.footer { grid-area: footer; }

CSS Grid est un outil extrêmement puissant pour créer des layouts complexes et responsives. Pratiquez ces concepts pour les maîtriser !
