Guide Complet de Flexbox

Apprenez progressivement à utiliser Flexbox à travers des exemples interactifs

1. Qu'est-ce que Flexbox ?

Flexbox est un modèle de mise en page CSS conçu pour créer des designs responsives plus efficacement. Il permet de distribuer l'espace et aligner des éléments dans un conteneur, même lorsque leur taille est inconnue ou dynamique.

.conteneur {
  display: flex;
}

Cette simple déclaration transforme un élément en conteneur flex et ses enfants directs en éléments flex.

2. Conteneur Flex et Éléments Flex

Le système Flexbox se compose de deux types d'éléments : le conteneur flex (parent) et les éléments flex (enfants).

3. Direction des éléments (flex-direction)

La propriété flex-direction définit la direction principale selon laquelle les éléments sont placés dans le conteneur.

1
2
3
.conteneur {
  display: flex;
  flex-direction: row;
}

4. Répartition des éléments (justify-content)

La propriété justify-content définit l'alignement le long de l'axe principal. Elle permet de distribuer l'espace entre et autour des éléments.

1
2
3
.conteneur {
  display: flex;
  justify-content: flex-start;
}

5. Alignement vertical (align-items)

La propriété align-items définit l'alignement des éléments le long de l'axe secondaire (perpendiculaire à l'axe principal).

1
2
3
.conteneur {
  display: flex;
  align-items: stretch;
}

6. Gestion du passage à la ligne (flex-wrap)

Par défaut, les éléments flex essaient de tenir sur une seule ligne. La propriété flex-wrap permet de changer ce comportement.

1

.conteneur {
  display: flex;
  flex-wrap: nowrap;
}

7. Propriétés des éléments flex

Les éléments flex enfants peuvent aussi avoir des propriétés spécifiques pour contrôler leur comportement.

flex-grow

Définit la capacité d'un élément à grossir si nécessaire.

Flexbox est un outil puissant pour créer des layouts modernes et responsives. Pratiquez ces concepts pour les maîtriser !
